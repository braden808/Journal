What is Responsive Design?

Responsive Design is a design that adjusts to size of the screen. In order to have a responsive design, em mused be used as well as media queries. Responsive design is used to make your website more flexible and apealing in any size or form.

Mobile design is the design to fit on a mobile device. This is very important because mobile devices are becoming very big and becoming the primary internet use for many people.
